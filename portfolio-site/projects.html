DDNS Updater w/automated Failover

This is my custom DDNS updater built for homelab accessibility and Cloudflare integration. This tool securely checks my public IP and updates DNS records automatically using Python and systemd scheduling. In order to ensure that the services I am hosting to the internet experience as little downtime as possible, I have included a falback script. This was designed to allow my T30 server to take over the responsibilities for this service should my Pi Cluster go offline. Designed with resilience, uptime and security in mind.

DDNS Updater with automation and Fallback scripting

I started on this project because I wanted to be able to host backend logic and LLM chatbots on my private server. Unfortunately, I do not have a static IP address. However, my Cloudflare account allows me to use a subdomain to resolve my public IP address. This means I am able to set up the front end applications to target a subdomain. The only issue remaining is that anytime my ISP updates my IP address it breaks everything until I manually update it in Cloudflare. This updater automates the entire process.

The first step I created a Python script that I could run on my server that would fetch my current public IP address, compare it to the existing DNS record on Cloudflare, and update it if there’s a mismatch. I broke the logic into modular components — one script to grab the IP, one to handle the Cloudflare API call, and one to load the config securely from a .env file. That way, I could keep credentials out of the codebase and make the whole thing portable across environments.

Once the core logic was working, I wrapped it in a main.py entrypoint and scheduled it using systemd .service and .timer units. This lets the updater run automatically at regular intervals without needing cron or manual triggers.

From there, I added a fallback script that checks if my Pi is online and running its own updater — if it is, the server disables its own instance to avoid collisions. If the Pi goes down, the server picks up the slack. Simple, resilient, and hands-off.
